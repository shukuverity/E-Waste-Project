# E-Waste-Project
A project based on E-waste management.
In Nairobi, electronic waste has become an environmental hazard. Rapid adoption of electronics and improper disposal habits have led to accumulation of electronics in households and small businesses. Disposal is done through informal channels that employ unsafe practices such as open-air burning. As electronics contain hazardous chemicals such as lead and mercury, open disposal practices threaten human health by causing disease as well as environmental degradation through pollution. Organizations like NEMA and certified e-waste recyclers have introduced regulations and recycling plants to counter this problem. However, the public still lacks efficient channels easily locate safe channels to dispose their electronic waste. Most of the existing collection systems target institutional clients only and there are also no digital systems that connect individuals to certified recyclers. For this reason, this project seeks to develop a web application that allows the public individuals to schedule electronic waste pickups, locate certified recyclers and earn credits for responsible disposal. The system will be designed using OOAD and will employ the Scrum process. Research will be conducted using existing literature, observation, and interviews. The system will finally be designed using modern web technologies and tested to ensure that it meets the desired goal.
